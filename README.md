
# Online Sales Dashboard - Power BI

## Project Overview
Developed an interactive Power BI dashboard to analyze online sales performance and provide actionable business insights.

## Tools Used
- Power BI
- Excel
- DAX
- Data Modeling

## Features
- Interactive filters and slicers
- Sales analysis by state, category, and customer
- Monthly profit trend analysis
- Category-wise and sub-category-wise performance tracking
- Dynamic visualizations and drill-down analysis

## Dashboard KPIs
- Total Sales Amount
- Total Profit
- Total Quantity Sold
- Average Order Value

## Visualizations Used
- Bar Chart
- Pie Chart
- Donut Chart
- Line Chart
- Area Chart
- Map
- Slicers

## Key Learnings
- Data Cleaning and Transformation
- Data Modeling and Relationships
- DAX Calculations
- Interactive Dashboard Design
- Business Insight Generation
